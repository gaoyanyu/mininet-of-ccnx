Mini-OF-CCNx: Mininet OpenFlow CCNx Prototyping Framework
========================================================

### What is Mini-OF-CCNx?

Mini-OF-CCNx is a Mini-CCNx fork that extend Mini-CCNx, a Mininet CCNx implementation, to support OpenFlow forwarding.
